Jump height

Estimates the highest point a jump reaches.

Formula

JumpHeight = LaunchVelocity^2 / (2 * Gravity)

This page breaks the formula down into what it is for, what each variable means, and whether it depends on any fixed constants.

Context

Why you would use it

Estimates the highest point a jump reaches.

Back to Movement formulas

Variables

What changes from use to use

LaunchVelocity is upward speed. Gravity is downward acceleration magnitude.

Constants

What stays fixed

2 is part of the standard motion equation.

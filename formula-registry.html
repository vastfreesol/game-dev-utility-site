Distance between two 2D points

Distance = sqrt((x2 - x1)^2 + (y2 - y1)^2)

Purpose: Measures how far one point is from another in 2D space.

Variables: x1, y1 and x2, y2 are the two points you are comparing.

Constants: None

Direction vector from A to B

Direction = normalize(B - A)

Purpose: Gives you a unit direction pointing from one point to another.

Variables: A is the start point. B is the target point.

Constants: None

Angle to target in 2D

Angle = atan2(y2 - y1, x2 - x1)

Purpose: Finds the facing angle from one point to another.

Variables: x1, y1 is the source. x2, y2 is the target.

Constants: None

Speed from distance and time

Speed = Distance / Time

Purpose: Finds how fast something must move to cover a distance in a set time.

Variables: Distance is how far it travels. Time is how long it takes.

Constants: None

Travel time

Time = Distance / Speed

Purpose: Shows how long movement takes when distance and speed are known.

Variables: Distance is the path length. Speed is movement rate.

Constants: None

Jump height

JumpHeight = LaunchVelocity^2 / (2 * Gravity)

Purpose: Estimates the highest point a jump reaches.

Variables: LaunchVelocity is upward speed. Gravity is downward acceleration magnitude.

Constants: 2 is part of the standard motion equation.

Time to apex

TimeToApex = LaunchVelocity / Gravity

Purpose: Shows how long the jump spends rising before it peaks.

Variables: LaunchVelocity is upward start speed. Gravity is downward acceleration magnitude.

Frame time from FPS

FrameTimeMs = 1000 / FPS

Purpose: Turns a framerate target into milliseconds per frame.

Variables: FPS is your target frames per second.

Constants: 1000 is the number of milliseconds in one second.

Open formula

Horizontal FOV from vertical FOV

HFOV = 2 * atan(tan(VFOV / 2) * AspectRatio)

Purpose: Converts a vertical camera FOV into a horizontal FOV.

Variables: VFOV is vertical field of view. AspectRatio is width divided by height.

Constants: 2 converts the half-angle math back into a full angle.

Open formula

Aspect ratio

AspectRatio = Width / Height

Purpose: Describes the shape of the screen or render target.

Variables: Width and Height are the screen or texture dimensions.

Constants: None

Texture memory

MemoryBytes = Width * Height * BytesPerPixel

Purpose: Gives a rough uncompressed memory cost for a texture.

Variables: Width and Height are texture dimensions. BytesPerPixel depends on the format.

Clamp

Clamp(Value, Min, Max)

Purpose: Forces a value to stay inside a safe range.

Variables: Value is the number you are controlling. Min and Max are the allowed edges.

Expected value

ExpectedValue = Sum(Outcome * Probability)

Purpose: Shows the average result you should expect over many rolls.

Variables: Outcome is the reward or result. Probability is its chance.

Constants: None

Open formula

At least one success

AtLeastOne = 1 - (1 - p)^n

Purpose: Shows the chance that repeated attempts succeed at least once.

Variables: p is single-attempt chance. n is number of attempts.

Constants: 1 represents the full probability range.

Open formula

Linear XP growth

XPForLevel = BaseXP + (Level - 1) * StepXP

Purpose: Builds a simple progression curve with a fixed increase per level.

Variables: BaseXP is the starting requirement. StepXP is the increase per level.

Constants: 1 keeps level one on the base XP value.

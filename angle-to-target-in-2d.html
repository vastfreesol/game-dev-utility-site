2D formulas

Angle to target in 2D

Use this when something in your game needs to face or aim toward a target in 2D, whether that is a turret, a sprite, a projectile launcher, or a UI indicator.

The formula

Angle = atan2(y2 - y1, x2 - x1)

Why you would use it

Direction tells you where to go. Angle tells you how to rotate so something points at the target. This page helps you check the value in both degrees and radians and see the facing line update live.

How to think about it

Face the target from a known axis

Most 2D angle calculations use the positive X axis as the zero-degree reference, then measure how far the target sits around that circle. That is why tan2 is such a common tool for aiming and facing logic.

Game-dev use cases

  • Rotate a turret or gun sprite toward the cursor.
  • Turn an enemy so it faces the player.
  • Drive arrow indicators and direction markers.

Variable breakdown

What each piece is doing

  • x2 - x1: the horizontal offset to the target.
  • y2 - y1: the vertical offset to the target.
  • atan2(dy, dx): returns the angle for that offset in a quadrant-safe way.

Common mistake

Degrees and radians are not interchangeable

Many engines expose rotation in degrees, while lower-level math functions return radians. Always check which one your engine API expects before applying the result.

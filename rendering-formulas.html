Frame time from FPS

FrameTimeMs = 1000 / FPS

Purpose: Turns a framerate target into milliseconds per frame.

Variables: FPS is your target frames per second.

Constants: 1000 is the number of milliseconds in one second.

Horizontal FOV from vertical FOV

HFOV = 2 * atan(tan(VFOV / 2) * AspectRatio)

Purpose: Converts a vertical camera FOV into a horizontal FOV.

Variables: VFOV is vertical field of view. AspectRatio is width divided by height.

Constants: 2 converts the half-angle math back into a full angle.

Aspect ratio

AspectRatio = Width / Height

Purpose: Describes the shape of the screen or render target.

Variables: Width and Height are the screen or texture dimensions.

Constants: None

Texture memory

MemoryBytes = Width * Height * BytesPerPixel

Purpose: Gives a rough uncompressed memory cost for a texture.

Variables: Width and Height are texture dimensions. BytesPerPixel depends on the format.

Constants: None

Pixel density

PixelsPerMeter = ResolutionPixels / WorldMeters

Purpose: Helps compare how much visual detail is spread across a world-space size.

Variables: ResolutionPixels is the texture or sprite size. WorldMeters is the in-game size.

Constants: None

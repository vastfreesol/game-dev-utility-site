2D formulas

Distance between two 2D points

Use this when you need to know how far apart two objects are in 2D space, whether that is a pickup radius, chase range, hit check, waypoint gap, or screen-space distance.

The formula

Distance = sqrt((x2 - x1)^2 + (y2 - y1)^2)

Why you would use it

Most of the time you do not care about the textbook name. You care about the practical question: how far away is object B from object A? This page gives you the formula, a visual plot, and a live result so you can reason about it quickly.

How to think about it

Measure the gap, not the pathing

This formula gives the straight-line distance between two points. In game terms, it is the raw gap between object positions before you factor in obstacles, navmesh routing, or path cost.

Game-dev use cases

  • Check whether a player is close enough to pick up an item.
  • Measure aggro range between an enemy and a target.
  • Estimate how far a projectile or unit is from a waypoint.

Variable breakdown

What each piece is doing

  • x1, y1: the first object's position.
  • x2, y2: the second object's position.
  • x2 - x1: the horizontal difference.
  • y2 - y1: the vertical difference.
  • sqrt(...): converts the squared gap into the final linear distance.

Common mistake

Distance is not direction

Distance only tells you how far apart the points are. If you need the direction to move toward the target, open the direction vector formula next.

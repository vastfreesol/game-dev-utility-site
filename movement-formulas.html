Speed from distance and time

Speed = Distance / Time

Purpose: Finds how fast something must move to cover a distance in a set time.

Variables: Distance is how far it travels. Time is how long it takes.

Constants: None

Travel time

Time = Distance / Speed

Purpose: Shows how long movement takes when distance and speed are known.

Variables: Distance is the path length. Speed is movement rate.

Jump height

JumpHeight = LaunchVelocity^2 / (2 * Gravity)

Purpose: Estimates the highest point a jump reaches.

Variables: LaunchVelocity is upward speed. Gravity is downward acceleration magnitude.

Constants: 2 is part of the standard motion equation.

Time to apex

TimeToApex = LaunchVelocity / Gravity

Purpose: Shows how long the jump spends rising before it peaks.

Variables: LaunchVelocity is upward start speed. Gravity is downward acceleration magnitude.

Lerp

Lerp(A, B, T) = A + (B - A) * T

Purpose: Moves some percentage of the way from one value to another.

Variables: A is the start. B is the end. T is the percentage.

Constants: None

Remap

OutMin + ((Value - InMin) / (InMax - InMin)) * (OutMax - OutMin)

Purpose: Converts a value from one range into another range.

Variables: Value is the input. InMin and InMax are the source range. OutMin and OutMax are the target range.

Constants: None

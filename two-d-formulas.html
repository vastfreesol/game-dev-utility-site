Distance between two 2D points

Distance = sqrt((x2 - x1)^2 + (y2 - y1)^2)

Purpose: Measures how far one point is from another in 2D space.

Variables: x1, y1 and x2, y2 are the two points you are comparing.

Constants: None

Direction vector from A to B

Direction = normalize(B - A)

Purpose: Gives you a unit direction pointing from one point to another.

Variables: A is the start point. B is the target point.

Constants: None

Angle to target in 2D

Angle = atan2(y2 - y1, x2 - x1)

Purpose: Finds the facing angle from one point to another.

Variables: x1, y1 is the source. x2, y2 is the target.

Constants: None

Pixels per unit

PPU = SpritePixels / WorldUnits

Purpose: Converts sprite size into a clean world scale.

Variables: SpritePixels is the sprite size. WorldUnits is the in-game size you want.

Constants: None

Tilemap world size

MapWidth = TileWidth * Columns; MapHeight = TileHeight * Rows

Purpose: Works out the final size of a tilemap from tile size and grid count.

Variables: TileWidth and TileHeight are tile dimensions. Columns and Rows are tile counts.

Constants: None

Sprite sheet frame count

Frames = floor(SheetWidth / FrameWidth) * floor(SheetHeight / FrameHeight)

Purpose: Shows how many frames fit in a sprite sheet.

Variables: SheetWidth and SheetHeight are the full sheet size. FrameWidth and FrameHeight are each frame size.

Constants: None

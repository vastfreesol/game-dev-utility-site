Core utility formula

Lerp between two values

Use lerp when a value needs to move part of the way from a start point to an end point.

At a glance

Lerp(A, B, T) = A + (B - A) * T

Lerp is a small formula that shows up everywhere in games, from UI bars and camera movement to audio fades and animation blending.

Simple explanation

What the formula means

Lerp starts at A, looks at B, then moves some percentage of the distance between them based on T.

Code example

float result = a + (b - a) * t;

When to use it

Common game-dev uses

  • Blend a UI fill amount toward a target value.
  • Move a camera or object smoothly from one position to another.
  • Fade volume, light intensity, or post-process values over time.

Variables

What each variable represents

  • A: The starting value.
  • B: The ending value.
  • T: How far to move from A to B. 0 means stay at A, 1 means go all the way to B.

Constants

Fixed values in the formula

  • No fixed constants. The behavior comes from the values you provide.
